Project Description

Gold Tips is a project designed to provide highly accurate future gold rates using the help of various ML techniques. Along with gold rate prediction, live gold, silver, and platinum rates are provided ( corresponding to both national and international markets) with interactive visualization tools.



Algorithms Description

1

Linear Regression

In statistics, linear regression is a linear approach to modelling the relationship between a dependent variable and one or more explanatory variables independent variables.

2

M.A.R.S

Multivariate adaptive regression splines, is a flexible regression method that automatically searches for interactions and non-linear relationships. MARS models can be thought of as linear models in a higher dimensional basis space. Each term in a MARS model is a product of so called “hinge functions”.

3

XG Boost

XGBoost is an optimized gradient boosting method, where multiple decision tree learners with high error rates are combined to form a strong learner.


4

Gaussian Naive Bayes

GNB is a variation of the naive bayes theorem that uses the prior conditional probability of the target feature given the dependent features’ occurance, but also assumes that each feature has no correlation to the target.
